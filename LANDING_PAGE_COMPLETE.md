### 1. View Landing Page
```bash
npm run dev
# Open http://localhost:5173/
```

### 2. Customize School Name
Edit `src/pages/LandingPage.jsx` (lines 85-87):
```jsx
<h1 className="font-bold text-lg">YOUR SCHOOL NAME</h1>
<p className="text-xs text-blue-100">YOUR TAGLINE</p>
```

```jsx

```

```bash

# Deploy to Vercel
```
